# 💄 Beauty Salon Web App

A modern beauty salon web application built with Vanilla JavaScript and Supabase.
This project provides a dynamic and user-friendly platform for managing salon services, specialists, and user interactions.

## 📌 Live Demo

🔗[view Demo](https://paria-rajaby.github.io/beauty-salon/)

## 🚀 Features

- 🔐 User authentication (Sign up / Login / Logout)
- 👩‍🎨 Display salon specialists dynamically
- 💇‍♀️ Services listing
- 📅 Booking-ready structure (expandable)
- 📱 Fully responsive design
- ⚡ Real-time database integration with Supabase
- 🧩 Modular vanilla JS structure

## 🛠️ Tech Stack

- Frontend: HTML, CSS, Vanilla JavaScript
- Backend & Database: Supabase
- Authentication: Supabase Auth
- Styling: CSS3
